## 公式结构

### 马来西亚 tab（列 A~O，菲/泰/新同构但无 LVG 列）

手动填写：A 产品名字、B 商品成本 RMB、C 摊销成本 RMB、D 实重 g、E 本地展示价。

| 列 | 含义 | 公式（以第 5 行为例） |
|----|------|---------------------|
| F | 毛利率 | `=IFERROR(N5/E5,"")` |
| G | 成本（进价+摊销）MYR | `=(B5+C5)/$B$2`（B2=汇率 1.6997） |
| H | 跨境物流成本 MYR | `=IFERROR(VLOOKUP(ROUNDUP(D5,-1),'物流价卡'!$D:$E,2,0),"")` |
| I | 佣金费 MYR | `=E5*$C$2`（C2=8.46%） |
| J | 交易手续费 MYR | `=E5*$D$2`（D2=3.78%） |
| K | SFP 项目费 MYR | `=IF(E5*$E$2>5,5,E5*$E$2)`（封顶 5 MYR） |
| L | 达人佣金 MYR | `=E5*$F$2`（F2=12%） |
| M | LVG 税 MYR | `=E5*10%` |
| N | 利润 MYR | `=E5-SUM(G5:M5)` |
| O | 利润 RMB | `=N5*$B$2` |

菲律宾/泰国/新加坡：无 LVG 列，M=利润（当地币）`=E5-SUM(G5:L5)`，N=利润 RMB `=M5*$B$2`，其余同构。SFP 封顶分别为 120 PHP / 199 THB /（新加坡费率空）。

### 越南 tab（特殊：按税前价计费，多图片列，列 A~P）

手动填写：B 产品名、C 商品成本 RMB、D 摊销成本 RMB、E 实重 g、F **税前价 VND**。

| 列 | 含义 | 公式 |
|----|------|------|
| G | 本地展示价 | `=(F5+J5)*(1+$G$2)`（G2=VAT 10%，J5=物流） |
| H | 毛利率 | `=IFERROR(O5/F5,"")` |
| I | 成本 VND | `=(C5+D5)/$C$2`（C2=0.00029） |
| J | 物流 VND | `=IFERROR(VLOOKUP(ROUNDUP(E5,-1),'物流价卡'!$J:$K,2,0),"")` |
| K | 佣金 | `=F5*$D$2`（3%，按税前价） |
| L | 手续费 | `=F5*$E$2`（5%，按税前价） |
| M | SFP | `=IF(G5*$F$2>30000,30000,G5*$F$2)`（按**展示价**计，封顶 30,000） |
| N | 达人佣金 | `=F5*$H$2`（10%，按税前价） |
| O | 利润 VND | `=F5-SUM(I5:N5)` |
| P | 利润 RMB | `=O5*$C$2` |
