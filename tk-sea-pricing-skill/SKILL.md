# TikTok Shop 东南亚定价计算器

基于用户的飞书定价参考表（《TikTok Shop东南亚跨境》TikTok Shop商品定价表参考，token: LC0lsFA07hUx0gtwMpOcSyqPnNb）的公式结构，对五个国家做定价与利润测算。

## 核心原则

1. **先收集，后计算**：必填信息缺一项就必须先问用户，绝不允许带着缺口开算。
2. **用户不确定的项给预估**：用户说"不知道/大概/你看着办"时，给一个合理预估值并明确说明这是预估、依据是什么，得到确认后使用。
3. **平台数据先核验再使用**：汇率、佣金、手续费、SFP、税、物流价格——参考表里的快照可能过时（尤其物流价格已确认过时），计算前要按下方「自动核验」流程拿最新值；核验不到的，用快照值兜底并在结果中明确标注"未核验，为参考表旧值"。
4. **物流成本 = 头程（前端）+ 尾程（尾端）两段之和**。参考表里的单段查表值是旧逻辑，不能直接用。

## 第一步：收集必填信息（手动输入项）

以下信息必须从用户处拿到，缺什么问什么（用 AskUserQuestion 或直接提问）：

| 项目 | 单位 | 说明 |
|------|------|------|
| 产品名 | - | 用于报告标题和确定商品类目（影响佣金率） |
| 商品成本 | RMB | 进货价 |
| 摊销成本 | RMB | 包装费 + 国内快递费 + 水电费等 |
| 实重 | g | 物流计费用，向上取整到 10g 档 |
| 本地展示售价 | 当地货币 | 马来西亚=MYR；其他国家用对应货币（越南填的是**税前价**，展示价由公式算出） |
| 目标国家 | - | 不说明时默认马来西亚，但要和用户确认 |
| **物流方式** | - | **官方小件运输** 或 **自行跨境运输**。选自行运输时追问两项：①头程+尾程合计预估快递费（RMB/单）②平均每单海外仓储费（RMB/单），物流成本=两者之和，不再查官方价卡 |

可选但影响结果的项（用户没提就问，问不到给默认值并标注）：
- **达人佣金率**：参考表默认马来西亚 12%、其他国家 10%。这是商家自己设置的，必须确认。
- **是否参加 SFP（超级运费券活动）**：参考表马来西亚当前 0%，其他国家 5%~6%。
- **马来西亚专属**：2025/9/13 起佣金区分超级补贴计划（BXP）商家与非 BXP 商家（费率差约 4 个点），问一下用户是否 BXP 商家；不知道就两种都算给对比。

**所有问题一轮问完**（一次 AskUserQuestion 最多 4 问，必要时合并选项），不允许多轮反复追问拖慢节奏。

## 第二步：取数（结果优先，核验后置）

**先出结果，再补精度。** 用户等待时间目标：收齐输入后 1 分钟内见到完整报价表。取数规则：

1. **默认直接用缓存+快照开算**：`references/verified-rates.md` 里 30 天内的核验值、`formulas.md` 快照值，拿来就算，逐项标注来源。**唯一必做的联网动作是汇率**（`curl https://open.er-api.com/v6/latest/{币种}`，1 秒出）。
2. **联网核验每项至多一跳一次**：只走缓存里的 knowledge_id 直链或 PDF 直链（Academy 文章匿名可读），一次拿不到立即用兜底值标注"⚠️待核验"，继续算。**禁止**：逐层点击寻路、为单个费率多轮浏览器交互、现场安装解析工具。
3. **哪里快搜哪里**：需要搜索时用最快渠道（搜索引擎 `site:seller.tiktokglobalshop.com`、权威跨境资讯均可），不局限于用户最初提供的网址。
4. **物流零联网**：物流费用一律按 `references/logistics.md` 用本地价表计算（用户提供的最新官方价卡），不查官网、不需要登录。
5. **输出后列"待精确项"**：把用了预估/旧值的项集中列出，用户有需要再单独深查，不拖累主流程。

**飞书参考表只是基础计算框架之一，不是费目全集。** 核验中发现官方存在参考表没有的费目（已知：菲律宾"订单处理费"、泰国"平台基础设施费"+"电商增长服务费"、越南"订单处理手续费"），**直接加进计算并标注，不需要请示用户**；费率拿不到就给保守预估值标注📊。今后发现新费目同样处理，并回写缓存。

**物流成本两种模式**（按第一步用户选择，公式与价表详见 `references/logistics.md`）：
- **官方小件运输**：总物流成本 = 头程（按实重 10g 档）+ 尾程（分区固定价，**分区不同价时取最高值**），直接用本地价表 `东南亚跨境物流运费价格表.xlsx` 计算。
- **自行跨境运输**：物流成本 = 用户提供的头程+尾程合计快递费 + 平均每单海外仓储费；尾程用户不清楚时可用本地 `海外仓价卡汇总260515.xlsx` 按最高价目的州估算。

**核验后回写缓存**：新核验值/新费目/新直链一律更新 `references/verified-rates.md`，让下一次计算零搜索。

## 第三步：计算

五国公式结构见 `references/formulas.md`（含每国列结构、费率快照、物流查表逻辑）。要点：

- **马来西亚/菲律宾/泰国/新加坡**（售价含税逻辑相同）：
  - 成本（当地币）=（商品成本 + 摊销成本）÷ 汇率（RMB per 1 当地币）
  - 各费用 = 展示售价 × 对应费率；SFP 有封顶（MY 5 MYR / PH 120 PHP / TH 199 THB）
  - 马来西亚多一项 LVG 税 = 售价 × 10%
  - 利润（当地币）= 展示售价 −（成本 + 物流 + 佣金 + 手续费 + SFP + 达人佣金 [+ LVG税]）
  - 毛利率 = 利润 ÷ 展示售价
- **越南**（特殊，按税前价计费）：
  - 用户提供**税前价 F**；展示价 =（F + 物流）×（1 + VAT 10%）
  - 佣金/手续费/达人佣金按 F 计；SFP 按展示价计、封顶 30,000 VND
  - 利润 = F −（成本 + 物流 + 佣金 + 手续费 + SFP + 达人佣金）；毛利率 = 利润 ÷ F
- 利润（RMB）= 利润（当地币）× 汇率

## 第四步：输出结果

输出一张完整明细表，必须包含计算涉及的**所有**类目，每行带：金额（当地币）、金额（RMB）、费率/依据、数据来源标注（✅已核验 / ⚠️参考表旧值 / 👤用户提供 / 📊预估）。表后给出：

- **毛利率**、**单件利润**（当地币 + RMB）醒目展示
- 核验情况说明（哪些核验成功、哪些用了旧值、核验时间）
- 如毛利率为负或低于 15%，主动提醒并给出建议售价区间

注意参考表里还有一项提示：**相关货损需卖家自行评估**，输出时提醒用户该表未含货损/退货成本。
